## 🚀 What this app does

Undercover helps keep AI-made code from standing out before you push it to a public repo. It reviews your code and commit text, then rewrites parts that can give away an AI tool or model. It is meant for people who use AI helpers and still want clean, natural-looking code and commit history.

Use it when you want to:
- Remove clear AI-style wording
- Clean up comments and variable names
- Smooth out commit messages
- Keep your open-source work looking human-written
- Review output before you publish it

- Repetitive naming

It can also help shape commit messages so they read more like normal developer commits. This is useful when you want the message to be short, clear, and less machine-like.

### 🔎 Pre-push review

Before you push, Undercover can scan the work one more time. This gives you a chance to catch anything that still feels out of place.
